RequiresExtendedMasterSecret()

This implementation supports RFC 7627 and will always negotiate the extended_master_secret extension where possible.

Declaration

Type Description
System.Boolean 
true
if the handshake should be aborted when the peer does not negotiate the

false

Remarks

When connecting to a peer that does not offer/accept this extension, it is recommended to abort the handshake. This option is provided for interoperability with legacy peers, although some TLS features will be disabled in that case (see RFC 7627 5.4).
