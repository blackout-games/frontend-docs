Class UnityPlatformAttribute

Use this attribute to define a specific set of platforms you want or do not want your test(s) to run on.

You can use this attribute on the test method, test class, or test assembly level. Use the supported enumeration values to specify the platforms. You can also specify which platforms to test by passing one or more RuntimePlatform values along with or without the include or exclude properties as parameters to the Platform attribute constructor.

The test(s) skips if the current target platform is:

  • Not explicitly specified in the included platforms list
  • In the excluded platforms list
